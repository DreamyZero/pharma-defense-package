# Следующий этап архитектуры

- Backend переведен на NestJS.
- Добавлены JWT-аутентификация, DTO и guard.
- Добавлена Prisma schema для PostgreSQL.
- Подготовлен контур для пользователей, аудита и администрирования.
- Docker Compose теперь включает PostgreSQL и Neo4j.

Этот шаг соответствует требованиям диплома по регистрации и авторизации, API-интеграции, гибридной БД PostgreSQL + Neo4j и контейнеризации.
