# Demo script for defense

## Цель показа
Показать, что система закрывает основные требования ВКР: сбор и унификация данных, поиск по синонимам, проверка взаимодействий, подбор аналогов, контекстные противопоказания, граф знаний, admin ETL и аудит.

## Сценарий на 7–10 минут
1. Проблема и решение.
2. Архитектура.
3. Вход в систему.
4. Admin ETL.
5. Пользовательские функции.
6. Данные и семантика.
7. Завершение.
